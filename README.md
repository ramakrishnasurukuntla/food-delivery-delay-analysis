# Food Delivery Delay Analysis

## Project Overview
This project analyzes the food delivery performance using SQL and Excel dashboards.

## Tools Used
  - SQL
  - Excel

## Analysis Performed
- Delayed order analysis
- Average delivery time analysis
- Restaurant revenue analysis
- city-wise delivery performance

## Dashboard Features
- KPI cards
- Pie charts
- Revenue Analysis Chart
- Delivery Time Analysis
